\doxysubsection{Detailed Description}
Goals service class. 

This class is used to create a node that subscribes to the topic /reaching\+\_\+goal/result and obtains the result of the action done by the user to send a goal. It also provides a service /goals\+\_\+service to provide the number of goals reached and cancelled.
