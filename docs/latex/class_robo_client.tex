\doxysubsection{Detailed Description}
Implements an action client, allowing the user to set a target position (x, y), or cancel it. 

This class is used to create a client node that sends goals to the action server defined in package assignment\+\_\+2\+\_\+2022. The user can input a goal position after entering \textquotesingle{}p\textquotesingle{} and pressing \mbox{[} E\+N\+T\+ER \mbox{]} and the robot will move to that position. The user can also cancel the current goal by entering \textquotesingle{}q\textquotesingle{}.
